# gulpfile

## gulp インストール

```
sudo npm install gulp --save-dev
```

## プラグイン

+ browser-sync
+ gulp-sass
+ gulp-plumber
+ gulp-pleeease

```
npm install --save-dev browser-sync gulp-pleeease gulp-plumber gulp-sass
```
