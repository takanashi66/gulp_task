# gulpfile

## プラグイン

+ browser-sync
+ gulp-sass
+ gulp-plumber
+ gulp-pleeease
+ gulp-notify
+ gulp-autoprefixer
+ gulp-sourcemaps
+ gulp-merge-media-queries
+ gulp-minify-css
+ gulp-uglify

### プラグインのインストール

```
npm install --save-dev gulp browser-sync gulp-plumber gulp-sass gulp-notify gulp-autoprefixer gulp-sourcemaps gulp-merge-media-queries gulp-minify-css gulp-uglify
```
